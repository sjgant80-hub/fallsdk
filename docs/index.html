Quick start

Three-line sovereign identity + peer channel:

import { load } from '@ai-native-solutions/estate';

const estate = await load();
const me = await estate.fallid.getOrCreate({ label: 'my node' });
const link = new estate.falllink.FallLink({ did: me.did });

link.on('message', ({ from, data }) => console.log(from, data));
await link.broadcast({ hello: 'estate' });

estate.load()

Fetches every library in parallel and returns a single object. Skip the ones you don't need with only.

import { load } from '@ai-native-solutions/estate';

// full estate
const all = await load();

// subset
const mini = await load({ only: ['fallid', 'foldkit'] });

// mirror base (e.g. self-hosted)
const local = await load({ base: 'https://mirror.example.org' });

// custom error handling
const est = await load({ onError: (name, err) => log(name, err) });

healthCheck()

HEAD every library and report status. Useful for a status page or CI probe.

import { healthCheck } from '@ai-native-solutions/estate';
const report = await healthCheck();
// [{ name:'foldkit', url:'…', ok:true, status:200 }, …]
